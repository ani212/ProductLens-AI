# ProductLens AI - Android Application 📱

This folder contains the codebase for the ProductLens AI Android client.

---

## Technical Specifications
- **Client runtime**: Native Kotlin/Java (Android SDK) or Cross-platform (Flutter / React Native).
- **Target Backend**: Connects to the Rust Rocket API hosted on Railway.

---

## Opening in Android Studio
1. Open **Android Studio**.
2. Select **Open an Existing Project**.
3. Choose the `/android-app` folder from this directory.
4. Sync Gradle and run the project emulator.
